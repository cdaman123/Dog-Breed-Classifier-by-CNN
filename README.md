# Dog-Breed-Classifier-by-CNN 
Project: Write an Algorithm for a Dog Identification App

We break this into separate steps. 

    Step 0: Import Datasets
    Step 1: Detect Humans
    Step 2: Detect Dogs
    Step 3: Create a CNN to Classify Dog Breeds (from Scratch)
    Step 4: Create a CNN to Classify Dog Breeds (using Transfer Learning)
    Step 5: Write your Algorithm
    Step 6: Test Your Algorithm
    
## Step 0 : Import data
Download [dog_dataset](https://s3-us-west-1.amazonaws.com/udacity-aind/dog-project/dogImages.zip) and [human_dataset](https://s3-us-west-1.amazonaws.com/udacity-aind/dog-project/lfw.zip) and unzip the data. 

    There are 13233 total human images.
    There are 8351 total dog images.
    
## Step 1 : Detect Humans

In this step I use `OpenCV` to detect human face. `OpenCV` provide many pretrained face detector and here we use  [ Haar feature-based cascade classifiers](https://docs.opencv.org/master/db/d28/tutorial_cascade_classifier.html).

## Step 2 : Detect Dogs
For identify dog face in image I used a pretrained Neural Network [`VGG-16`](https://pytorch.org/vision/stable/models.html) that have been trained on `ImageNet`, a very large, very popular dataset used for image classification and other vision tasks. `ImageNet` contains over 10 million URLs, each linking to an image containing an object from one of `1000 categories`.

## Step 3 : Create a CNN to Classify Dog Breeds (from Scratch)
I make a neural network Architecture using `Pytorch` and train that on labeled data set. Use `CrossEntropyLoss` as loss function and `Stochastic gradient descent(SGD)` as optimizer with learning rate `0.05`. Train model using `20 Epoch`. 

### Architecture of NN:
    Net(
    (conv1): Conv2d(3, 32, kernel_size=(3, 3), stride=(2, 2), padding=(1, 1))
    (conv2): Conv2d(32, 64, kernel_size=(3, 3), stride=(2, 2), padding=(1, 1))
      (conv3): Conv2d(64, 128, kernel_size=(3, 3), stride=(1, 1), padding=(1, 1))
    (pool): MaxPool2d(kernel_size=2, stride=2, padding=0, dilation=1, ceil_mode=False)
    (fc1): Linear(in_features=6272, out_features=500, bias=True)
    (fc2): Linear(in_features=500, out_features=133, bias=True)
    (dropout): Dropout(p=0.3)
    )
    
### Test result:
    Test Loss: 3.720266
    Test Accuracy: 13% (117/836)

## Step 5: Write Algorithm

For this I make a algorithm for predicting output:
* if a **dog** is detected in the image, return the predicted breed.
* if a **human** is detected in the image, return the resembling dog breed.
* if **neither** is detected in the image, provide output that indicates an error.
